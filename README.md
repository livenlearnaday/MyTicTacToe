MyTicTacToe

Setup requirement:
- Java 8
- Min Android Studio to use is Android Studio Koala that support gradle AGP 8.6.1


Components used:
- jetpack compose
- koin di
- voyager for compose navigation
- room for local database
- accompanist for permission request
